## ElasticSearch Demo

### Pre-Requisites

```bash
# Install ElasticSearch
brew install elasticsearch

# Verify ElasticSearch is Installed
elasticsearch --version

# Start ElasticSearch
elasticsearch
```

#### ElasticSearch URL
```
http://localhost:9200

```
